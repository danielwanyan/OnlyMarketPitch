## 判定逻辑

视频内容只有营销叫卖话术，完全没有商品信息介绍 → 命中

### 营销叫卖关键词

- 价格类：XX元、只要XX、特价、优惠价
- 促销类：买一送一、限时优惠、活动今天截止
- 包邮类：包邮、免运费
- 稀缺类：即将断货、库存紧张、最后XX件

### 不构成纯营销叫卖（有商品信息）

- 商品功能介绍
- 商品特点/优势说明
- 使用方法演示
- 成分/材质说明
