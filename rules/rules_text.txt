# 纯营销叫卖判定规则

## 版本信息

- 当前版本：2025-05-20-v2
- 上次更新：2025-05-20
- 更新内容：移除 product_title，区分纯ASR/纯OCR/ASR+OCR营销叫卖

---

## 规则索引

| 编号 | 规则名称 | 说明 |
|------|---------|------|
| MP01 | 纯营销叫卖 | 视频内容只有营销话术，无商品信息介绍 |

---

## 适用范围

本规则适用于 **AIGC 内容** 和 **非AIGC（真人）内容**。

根据 `aigc_video_id` 区分：
- `aigc_video_id` 有值 → AIGC纯营销叫卖
- `aigc_video_id` 为 `NULL` → 非AIGC纯营销叫卖

---

## 输入信息

### 文本信息（客户可见）

| 字段 | 说明 |
|------|------|
| ASR | 视频语音转文本 |
| OCR | 视频画面中识别出的文字 |
| Comment | 视频评论/描述 |
| Product_category | 产品分类（三级类目） |

**注意**：`Product_title`（产品标题）客户不可见，**不得作为判断依据**。

### 外部数据打标（权威数据，必须完全信任）

| 字段 | 说明 |
|------|------|
| `aigc_video_id` | AIGC 视频 ID |

**AIGC 判断逻辑**：
- `aigc_video_id` 有值（如 `7639939328541592854`）→ AIGC 内容
- `aigc_video_id` 为 `NULL` → 非AIGC（真人）内容

---

## 判定依据

**仅基于 ASR 和 OCR 进行判断**，不得使用 Product_title。

### 数据来源分类

| 情况 | ASR | OCR | 判定类型 |
|------|-----|-----|---------|
| 1 | 有营销话术 | 为空/无数据 | 纯ASR营销叫卖 |
| 2 | 为空/无数据 | 有营销话术 | 纯OCR营销叫卖 |
| 3 | 有营销话术 | 有营销话术 | ASR+OCR营销叫卖 |

---

## 规则详情

### MP01：纯营销叫卖

**定义**：视频内容只有营销叫卖话术，完全没有商品信息介绍。

**触发条件**（必须同时满足）：
1. ASR 或 OCR 中只有营销叫卖话术
2. 完全无商品功能、特点、使用方法等信息介绍

**营销叫卖关键词**：

| 类别 | 关键词 |
|------|--------|
| 价格类 | XX元、只要XX、特价、优惠价、多少钱、价格、只需、仅需、price、$、¥ |
| 促销类 | 买一送一、买赠、限时优惠、活动今天截止、错过再等一年、限时、最后机会、promotion、sale、discount |
| 包邮类 | 包邮、免运费、free shipping |
| 稀缺类 | 即将断货、库存紧张、最后XX件、手慢无、限量、抢完即止、limited stock |
| 购买引导 | 不买后悔、错过就没了、赶紧买、下单、购买、抢购、buy now、order now、shop now、add to cart、点击购物车 |

**商品信息介绍（有以下内容则不构成纯营销叫卖）**：
- 商品功能介绍
- 商品特点/优势说明
- 使用方法演示/说明
- 成分/材质说明
- 规格/参数介绍
- 效果展示（非营销话术）
- 适用场景说明

**判定逻辑**：

```
分析 ASR 和 OCR 内容
    ↓
是否只有营销叫卖话术？
    ├─ 是 → 命中 MP01（纯营销叫卖）
    └─ 否 → 无问题
```

**判定标准**：

| 情况 | 结果 |
|------|------|
| 只有营销话术，无任何商品信息 | 命中 |
| 营销话术 + 商品功能介绍 | 不命中 |
| 营销话术 + 使用方法演示 | 不命中 |
| 营销话术 + 成分/材质说明 | 不命中 |
| 营销话术 + 规格/参数 | 不命中 |

**细分类型**：

| 类型 | 说明 |
|------|------|
| 纯ASR营销叫卖 | 只有 ASR 有营销话术，OCR 为空或无营销内容 |
| 纯OCR营销叫卖 | 只有 OCR 有营销话术，ASR 为空或无营销内容 |
| ASR+OCR营销叫卖 | ASR 和 OCR 都有营销话术 |

**问题类型区分**：
- AIGC 内容 → AIGC纯营销叫卖（纯ASR/纯OCR/ASR+OCR）
- 非AIGC 内容 → 非AIGC纯营销叫卖（纯ASR/纯OCR/ASR+OCR）

---

## 判定示例

### 示例 1：命中（纯ASR营销叫卖）

**ASR内容**：
"这款产品只要99元！买一送一！今天下单还包邮！库存有限，手慢无！错过再等一年！"

**OCR内容**：
（为空或无数据）

**判定**：命中 MP01
**细分类型**：纯ASR营销叫卖
**理由**：只有价格、促销、包邮等营销话术，无任何商品功能、特点介绍

---

### 示例 2：命中（纯OCR营销叫卖）

**ASR内容**：
（为空或无数据，只有背景音乐）

**OCR内容**：
"限时特价！买一送一！最后100件！手慢无！"

**判定**：命中 MP01
**细分类型**：纯OCR营销叫卖
**理由**：OCR只有营销话术，ASR为空

---

### 示例 3：命中（ASR+OCR营销叫卖）

**ASR内容**：
"这款产品只要99元！买一送一！赶紧下单！"

**OCR内容**：
"限时特价！最后机会！点击购物车立即购买！"

**判定**：命中 MP01
**细分类型**：ASR+OCR营销叫卖
**理由**：ASR和OCR都只有营销话术，无商品信息

---

### 示例 4：不命中（有商品信息）

**ASR内容**：
"这款产品只要99元！买一送一！它采用了最新的防水技术，适合户外使用，续航可达8小时..."

**OCR内容**：
"防水等级：IPX7 | 续航：8小时 | 重量：150g"

**判定**：不命中
**理由**：虽然有营销话术，但包含了"防水技术"、"续航8小时"等商品信息介绍
